Random Tools I made for ttrpgs

Mutant Year Zero

Character Creator

A character creator I made for the Mutant Year Zero ttrpg.
Currently only saves to cookies. Not extensively tested.
MYZ Character Creator


Skills Cheatsheet

A list of the skills, role-specific skills separated.
Contains descriptions, separated by headings.
MYZ Skills Cheatsheet


Talents Cheatsheet

A list of the talents, role-specific talents in separate lists.
Contains descriptions.
MYZ Talents Cheatsheet


Mutations Cheatsheet

A list of the mutations, with descriptions.
MYZ Mutations Cheatsheet


Starfinder

Starship Creator

A tool to make starships for starfinder. Only saves to cookies currently.
Can print to pdf to save a statblock.
Starfinder Starship Creator


NPC Creator

A tool to make pretty basic NPCs pretty quickly. Doesn't follow the Alien Archive rules exactly (or much at all), and doesn't have the capacity to add special abilities, senses, or magic. You gotta keep that shit in your own head (or use something else).
You can explort individual NPCs to strings to be re-imported (which is why there's a lot of stuff missing, I had to hand-code the export function and I'm not made of time).
Starfinder NPC Creator (not RAW)


Shadowrun 5e

Random Run Generator

A tool I made for a home game to randomly generate run ideas that were a bit more detailed than just rolling on the tables in the book. So there are some specific references in in, like the Lab (which is the fixer's bar) and probably some gangs that are specific to my setting. It still works though, and I might go back and fix that at some point.
Random Run Generator


Run Rewards Calculator

A little calculator I made that works out what the rewards for a run should be. The session counter is in there because my games were slapdash as fuck and there were a lot of one-shots. But it's basically got everything.
Run Rewards Calculator


Core Weapon List

Another tool I made to help with my slapdashery. Just a list of the different weapons in the core book in case you're just making shit up and need to give some NPCs some specific equipment. Another one that I might come back and fix at some point to make a bit more useable, but it's all good now.
Core Weapon List


Sprite Creator/Cheatsheet

A tool I made for our technomancer so that she could keep track of sprites and also reference abilities and such. Didn't see much use, cause we worked out a better way, but maybe you haven't yet.
Sprite Maker


Magic Cheatsheet

This is not a spell list.
It's basically a slightly annoying, interactive list of the different types of magic, and related rolls and shit like that.
Magic Cheatsheet
